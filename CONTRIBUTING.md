# Contributing to OpenGov AI

Thank you for your interest in contributing! This project is in early development.

## Getting Started

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -am 'Add your feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a Pull Request

## Development Setup

- **Discovery / Ingestion / Response**: Go 1.22+
- **Risk Engine**: Rust 1.75+ (stable)
- **Dashboard**: Node.js 20+, npm
- **Infrastructure**: Docker, NATS

## Code Style

- Go: `gofmt` + `golangci-lint`
- Rust: `rustfmt` + `clippy`
- TypeScript: ESLint + Prettier

## Design Partners

We're actively looking for enterprise design partners. If your organization is dealing with AI agent governance challenges, please open an issue or reach out.
